# 📰 News Tracker

Follow news stories as they develop over days, months, and years.

## Stack
- **Frontend:** Next.js 14 + Tailwind (Vercel)
- **Backend:** FastAPI (Railway)
- **Database:** PostgreSQL + pgvector (Railway)
- **Sources:** RSS feeds + SearXNG
